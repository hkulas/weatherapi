WeatherAPI Documentation

Overview

The WeatherAPI is a RESTful service for saving and retrieving weather forecasts.

Setting Up

The application requires a WeatherAPI key to function. After obtaining a key, it should be added to the application.yml file in the following format: 


        weather-api:
          key: YOUR_WEATHER_API_KEY

Running the application locally


	git clone https://github.com/username/repository.git
	cd repository
	mvn clean install
    ./mvnw spring-boot:run

API Endpoints

POST /api/weather/forecast/{city}

Saves forecasts for today and tomorrow for chosen city.

Responses:

  • 200: Successfully saved forecasts.
  • 404: The city not found.
  • 409: Forecast for the city for today and tomorrow already exists.

GET /api/weather/forecast/{city}

Get a list of forecasts for today and tomorrow for chosen city.

Responses:

  • 200: Successfully retrieved a list of todays and tomorrows forecasts.
  • 404: A forecast for city was not found.

GET /api/weather/forecast

Get a list of all available forecasts.

Responses:

  • 200: Successfully retrieved a list of forecasts.
  • 404: No forecasts found.
